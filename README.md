# Installation

Clone repository:

```bash
git clone https://github.com/aslam8801/aslam-portfolio.git
```

Move into project:

```bash
cd aslam-portfolio
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Open:

```txt
http://localhost:3000
```
